# semana-omnistack-10
Projeto realizado durante a semana omnistack 10 (RocketSeat),  "Dev Radar".

Esse projeto foi desenvolvido com as seguintes tecnologias:

* Node.js
* React
* React Native
* Expo
* MongoDB
